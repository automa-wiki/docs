# 循环元素模块

遍历匹配选择器的元素。此模块与具有“元素”选项的[循环数据模块](/blocks/loop-data.md)相同，但此模块具有更多功能。

- **循环 ID** <br />
  用于标识循环的 ID。使用此 ID 来[引用循环的数据](#accessing-data)，或当使用[循环中断点](/blocks/loop-breakpoint)模块时。

- **元素选择器** <br />
  要循环遍历的元素的[元素选择器](../workflow/element-selector.md)。

- **最大数据量** <br />
  设置循环元素的限制。

- **加载更多元素** <br />
  一个加载更多元素的动作
	- **无**
	- **点击元素加载更多**
	- **点击链接加载更多**
	- **向下滚动加载更多**
