# 获取标签页 URL 模块

获取标签页的 URL。

- **选择标签页** <br>
  选择是获取当前活动标签页的 URL 还是所有标签页的 URL。

- **按键** <br>
  你想要模拟的按键或按键组合。

- **分配给变量** <br>
  是否将 URL 分配给[变量](../workflow/variables.md)。

- **变量名** <br>
  分配 URL 的变量的名称。

- **插入到表中** <br>
  是否将 URL 插入到[表](../workflow/table.md)中。

- **选择列** <br>
  URL 将被插入的列。
