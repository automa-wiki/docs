# 屏幕截图模块

对活动标签页进行屏幕截图。

- **截图对象** <br>
  选择是对页面、完整页面还是某个元素进行截图。

- **CSS选择器** <br>
  需要截图元素的CSS选择器。

- **保存截图到电脑** <br>
  是否将截图结果保存到电脑。

- **文件名** <br>
  截图文件的名称。

- **文件类型** <br>
  截图文件的文件类型，默认为`PNG`。

- **插入截图到表** <br>
  截图的[data URL](https://developer.mozilla.org/en-US/docs/Web/HTTP/Basics_of_HTTP/Data_URIs)是否被插入到[表](../workflow/table.md)中。

- **分配至变量** <br>
  是否将数据URL分配给[变量](../workflow/variables.md)。

- **变量名** <br>
  分配数据URL的变量名称。
