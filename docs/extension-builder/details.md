# 扩展程序：详情

在这个页面上，您将找到关于扩展程序的详细信息，例如名称、描述、图标和版本历史。您也可以在这里更改所有这些信息。
- **扩展程序图标**<br/>
  当用户在浏览器上安装您的扩展程序或者您将其发布到Chrome网上应用店时，这个图标将会显示出来。
  您想要设置为图标的图片必须是PNG格式的，且大小小于100KB。
- **扩展程序名称**<br/>
  扩展程序的名称，必须超过三个字符。
- **扩展程序描述**<br/>
  扩展程序的描述，这个字段是可选的。
- **版本历史**<br/>
  版本列表。您可以通过点击该版本上的下载按钮来下载特定版本的扩展程序。
