# 安装扩展程序

您从 Automa Chrome 扩展生成器控制面板下载的扩展 zip 文件，可以安装在基于 Chromium 的浏览器上，例如 Google Chrome、Microsoft Edge、Brave 等。

要将扩展安装到这些浏览器之一，请执行以下操作：
1. 打开 `chrome://extensions` 页面
2. 在该页面上，如果您使用的是 Google Chrome，请在右上角启用“开发者模式”开关。
3. 将扩展 zip 文件拖放到该页面中。
